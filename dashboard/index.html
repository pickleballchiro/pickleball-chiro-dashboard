🏓 One-time setup needed

Your dashboard needs a Google Sheets API key to read your data. This takes about 2 minutes.

  1. Go to console.cloud.google.com
  2. Open your existing project "My First Project"
  3. APIs & Services → Credentials → + Create Credentials → API Key
  4. Click the new key → Restrict to Google Sheets API only
  5. Copy the key and open dashboard/js/config.js
  6. Replace YOUR_API_KEY_HERE with your key and save
  7. Refresh this page
